A terminal based snake game built with [bubbletea](https://github.com/charmbracelet/bubbletea) and [lipgloss](https://github.com/charmbracelet/lipgloss)

# Usage

Just download the binary and run. You can set a custom width and height with `-width` and `-height`, the default values are 100 and 20.

## Keybinds

Control the snake with `arrow keys`, `wasd` or `hjkl`, restart the game with `r` and quit with `q`
